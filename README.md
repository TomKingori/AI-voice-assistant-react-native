# AI-voice-assistant-React-Native

## Get Started

install dev dependencies

```
npm install
```

or

```
yarn install
```

## Get API KEY
1. Go to https://openai.com<br/>
2. Create a new account and get your api key<br/>
3. Add the api key in constants/index.js file
 
## Run The App
```
npx react-native start
```
